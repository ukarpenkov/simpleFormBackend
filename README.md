# Настройка Django проекта

Это руководство поможет вам настроить и запустить Django проект на вашем локальном компьютере.

## Предварительные требования

- Установленный Python.
- Инструмент для работы с виртуальными окружениями (опционально, но рекомендуется).
- Установленный Django и необходимые зависимости.

## Установка

1. Клонируйте репозиторий на ваш локальный компьютер:

   ```bash
   git clone https://github.com/ukarpenkov/simpleFormBackend
   ```

2. Перейдите в директорию проекта:

   ```bash
   cd djangoProject/backend
   ```

3. Создайте и активируйте виртуальное окружение (опционально, но рекомендуется):

   ```bash
   python -m venv env
   source env/bin/activate  # Для Windows: env\Scripts\activate
   ```

4. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск сервера

1. Примените миграции для настройки базы данных:

   ```bash
   python manage.py migrate
   ```

2. Запустите сервер разработки:

   ```bash
   python manage.py runserver
   ```

3. Откройте API по адресу:
   ```
   http://127.0.0.1:8000/api/workplaces/
   ```

## Примечания

- Убедитесь, что пакет `django-cors-headers` установлен и правильно настроен, если вы обращаетесь к API с другого источника.
- Для устранения проблем проверьте файл `settings.py` на наличие конфигурации CORS.
